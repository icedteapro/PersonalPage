1. Angular: npm install -g @angular/cli
2. Install extension for Visual Studio Code "angular essentials", "material icon theme"
3. Add UI Material (same Bootrap): ng add @angular/material.
4. Add ExpressJs: "npm install --save express"
5. Install nodemon "npm install -g nodemon"
6. Run server: "nodemon server.js"
7. Install Cros: "npm install cros --save"
7. Install Body-Parser "npm install --save body-parser": Đây là lớp trung gian, xỷ lý JSON, text và mã hóa URL.
8. Install "npm install --save mongoose":  là một công cụ mô hình hoá đối tượng MongoDB, được thiết kế để làm việc trên môi trường bất đồng bộ.
9. Connect to MongoDB by CMD "Capture.PNG"
10. validator model (mongoose) "npm install --save mongoose-unique-validator".
11. EnCrypt password "npm install --save bcrypt"
12. Token for server "npm install --save jsonwebtoken"
13. Toast [alert] "npm install ngx-toastr --save" & "npm install @angular/animations --save"
14. Capcha "npm install ngx-captcha"
15. html "npm install jade"
16. Email "npm install express-mailer --save"
